# CDC Training

# Data Science and Machine Learning Practice Repository

This repository contains hands-on Python, Data Science, and Machine Learning practice notebooks developed as part of the CDC Training Program.  
It focuses on data cleaning, data analysis, visualization, and basic machine learning models using real-world datasets.

# Project Objectives

- Learn Python for Data Science
- Perform data cleaning and preprocessing
- Practice NumPy and Pandas
- Visualize data using Matplotlib and Seaborn
- Build basic Machine Learning models
- Work with real-world datasets

# Notebook Overview

## Data Cleaning and Analysis
- Data_Cleaning.ipynb – Handling missing values and preprocessing
- clustering.ipynb – Clustering techniques

## Machine Learning Projects
- Loan Predictor – Loan eligibility prediction
- Crop Predictor – Crop recommendation system
- House Price Dataset – Regression practice

## Python Libraries Practice
- NumPy Examples
- Pandas Examples
- Matplotlib Examples
- Seaborn Examples
- Scikit-learn Examples
- OpenCV Example

# How to Run

1. Clone the repository

git clone https://github.com/thoshhhh/Cdc-Traning.git  
cd Cdc-Traning

2. Install required libraries

pip install numpy pandas matplotlib seaborn scikit-learn opencv-python

3. Run Jupyter Notebook

jupyter notebook
